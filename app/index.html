Introduction

This visualization displays the financial trends of various companies. It utilizes dc.js and crossfilter.js to create charts, and processing to create the ball visualization.

We make HTTP requests to Quandl, an online repository for financial and economic data. Using this data, we construct interactive charts and predict investments.
